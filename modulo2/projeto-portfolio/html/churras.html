Projeto: Churrascômetro

Descrição:

Criei esse projeto com intuito de treinar alguns conceitos de html, css e javascript,
além de criar um ferramenta que auxiliará uma pessoa a realizar um churras de uma
forma mais simples, através do uso dessa ferramenta você poderá adquirir de forma mais
fácil os produtos necessários para fazer um churrasquinho em família, basta informar a
duração, quantidade de adultos e de crianças, e o Churrascômetro fará sua mágica.


    Tecnologias Usadas:

  • javascript
  • Html
  • Css
